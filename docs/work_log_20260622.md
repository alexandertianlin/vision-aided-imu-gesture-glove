## Key Debugging Steps

1. **Diagnostic log** → revealed `APPLY_FAILED` for ring finger
2. **Layer analysis**: vision receiver → HandMotionManager → FingerSolver
3. **IsCalibrated check** blocked all processing without IMU → bypassed
4. **ApplyVisionAnchorPose** returned false without IMU → added no-IMU path
5. **Ring finger stuck** in FIST position after takeover → added relaxation code
